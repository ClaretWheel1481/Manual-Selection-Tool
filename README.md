# Manul-Selection-Tool-Python-Version
###
针对CSGO搬砖手动选择商品的利润率计算器(USD TO CNY),由Python编写。
###
(对Python的知识几乎为0，故大部分代码十分臃肿,后续将不断学习优化)
###
注意：必须在可正常使用互联网的状态下使用，否则将无法启动(用于查询当前时间的USD TO CNY汇率)。
